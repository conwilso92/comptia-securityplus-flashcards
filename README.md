# CompTIA Security+ (SY0-701) Flashcard Study App

A browser-based flashcard app designed for studying the CompTIA Security+ certification exam. 256 cards covering all 5 exam domains: General Security Concepts, Threats Vulnerabilities & Mitigations, Security Architecture, Security Operations, and Security Program Management.

## How to use

- Tap a card to reveal the answer
- Rate yourself Missed, Shaky, or Got It to track your progress
- Filter by category to focus on one domain at a time
- Use the Missed and Shaky filters to drill your weak spots
- Shuffle to randomize the order

All acronyms include the full name underneath the answer. No login required — works on any device with a browser.
